# 🎬 Movie Agent AI

An AI-powered movie recommendation system built using Python, Gemini AI, and OMDb API.

## 🚀 Features

- 🎬 Search movies using OMDb API
- 🤖 AI-powered movie recommendations using Gemini
- ⭐ Get IMDb ratings, genres, directors, and movie plots
- 💬 Ask movie-related questions in natural language

## 🛠️ Technologies Used

- Python
- Google Gemini API
- OMDb API
- Requests Library
- python-dotenv
